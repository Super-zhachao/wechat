# wechat

安卓大作业，仿微信，简单UI，__未对接后台__，因技术有限，当初为了交作业现学现卖，做了两晚上的成果，代码可能有点粗糙，仅供参考（作业），__可能无法应用于实际项目__，曾经被大神级学长 评价为 __“老人版微信”__

## 开发环境

> Android Studio

> jdk1.8

>安卓版本,抱歉，忘记了，这项目很久了

## 使用教程

把代码克隆到本地，用AS打开项目，按照提示安装依赖（可能需要科学上网），OK
碰到问题可以骚扰：email:lcujn@qq.com
